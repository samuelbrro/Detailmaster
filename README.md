# Detail Master - Estética Automotiva

Site completo gerado a partir do material original.

## Estrutura de Arquivos

```
detailmaster/
├── index.html          ← Página principal
├── css/
│   └── style.css       ← Estilos completos
├── js/
│   └── script.js       ← Interatividade (navbar, animações)
└── images/             ← Pasta para imagens futuras
```

## Como usar

1. Abra o arquivo `index.html` em qualquer navegador
2. Para hospedar, faça upload de todos os arquivos mantendo a estrutura de pastas

## Personalizar

- **Cores**: edite as variáveis CSS no início do `style.css` (`:root { --red: #e01b24 ... }`)
- **Links de agendamento**: substitua os `href="#"` nos botões "Clique Aqui" pelos links reais (WhatsApp, Google Maps, Instagram, etc.)
- **Logo**: o logo atual é um SVG desenhado inline no `index.html`. Pode ser substituído por uma imagem real na pasta `images/`

## Tecnologias

- HTML5 semântico
- CSS3 com variáveis customizadas e animações
- JavaScript vanilla (sem dependências)
- Google Fonts: Bebas Neue, Rajdhani, Caveat, Inter
